# トレイアイコンツール（TrayIcon）

これは VB.NET で作成された Windows トレイ常駐アプリケーションです。起動時にタスクトレイにアイコンを表示し、CSVファイルに基づいてメニューを動的に生成します。

## 主な機能

- CSVファイルからメニュー構造を読み込み
- メインメニュー / サブメニュー / コマンドの3階層構造に対応
- メニュー項目のクリックで：
  - クリップボードにコピー
  - Ctrl+左クリック：パスやURLを開く
  - Ctrl+右クリック：内容をキー入力として送信（SendKeys）

## 使い方

1. コンパイル済みの EXE を任意のフォルダに配置。
2. 同じフォルダまたはサブフォルダに `.csv` ファイルを配置。
3. CSVファイルの形式は以下の通り：

メインメニュー,サブメニュー,表示名,内容


4. 起動するとタスクトレイにアイコンが表示されます。
5. 右クリックでメニューの再読み込みや終了が可能です。

## 注意点

- CSVはUTF-8で保存してください。
- 内容列は空にしないでください。
